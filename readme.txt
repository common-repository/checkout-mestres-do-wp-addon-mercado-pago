== Description ==

Personaliza os campos do Mercado Pago no Checkout Mestres do WP.

== Installation ==

= Instalação automática pelo Wordpress =

1. Selecione **Add New** em seu WordPress na aba **Plugins**.
2. Procure por **Checkout Mestres do WP Addon Mercado Pago**.
3. Clique em **Install Now** e **Activate Plugin**.

= For manual installation via FTP: =

1. Faça o upload do **checkout-mwp-addon-mp** na pasta **/wp-content/plugins/**.
2. Ative na seção **Plugins** do seu Wordpress.
